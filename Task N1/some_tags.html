Rare tags

Есть множество тегов, которые редко используются, но которые знать не помешает.

Вот они:

  • dl, dt, dd
  • pre
  • code, tt
  • kbd
  • var
  • sup, sub
  • small, big
  • blockquote, q
  • ins, del
  • abbr
  • address
  • nobr, wbr

dl, dt, dd

Описание:

Тег dl входит в тройку элементов dl, dt, dd, предназначенных для создания списка определений. Каждый такой список начинается с контейнера dl, куда входит тег dt создающий термин и тег dd задающий определение этого термина.

Пример тегов dl, dt, dd:

Термин 1
Определение термина 1
Термин 2
Определение термина 2

pre

Описание:

Элемент pre определяет блок предварительно форматированного текста. Такой текст отображается обычно моноширинным шрифтом и со всеми пробелами между словами. По умолчанию, любое количество пробелов идущих в коде подряд, на веб-странице показывается как один. Тег pre позволяет обойти эту особенность и отображать текст как требуется разработчику.

Пример тега pre:

			 -                -----  
			 -----           |-----
			||----|          ----||  
			||-----         -----||  
			||-----|       |-----||
			|| -----       ------||
			||- ----|     |------||
			||---||--     -------||
			||--|| --|   |-------||
			|| -|| |--   --- - --||
			|| -||  --|-|--| - ---|
			|---||  |-----| |-----|
			|---||   |----  |-----| 
			|----|    ---   |-----|
			|-----          ------|

code, tt

Описание:

Тег code предназначен для отображения одной или нескольких строк текста, который представляет собой программный код. Сюда относятся имена переменных, ключевые слова, тексты функции и т.д. Браузеры обычно отображают содержимое контейнера code как моноширинный текст уменьшенного размера.

Пример тега code:

function checkParent (src, dest) {
while (src != null) {
if (src.tagName == dest) return src
src = src.parentElement
}
return null
}

Пример тега tt:

Lorem ipsum dolor sit amet

kbd

Описание:

Тег kbd используется для обозначения текста, который набирается на клавиатуре или для названия клавиш. Браузеры обычно помечают текст в контейнере kbd моноширинным шрифтом.

Пример тега kbd:

<Alt>+<Backspace>

var

Описание:

Тег var используется для выделения переменных компьютерных программ. Браузеры обычно помечают текст в контейнере var курсивным начертанием.

Пример тега var:

content_title

sup, sub

Описание:

Тег sup отображает шрифт в виде верхнего индекса. Шрифт при этом отображается выше базовой линии текста и уменьшенного размера. Тег sub Отображает шрифт в виде нижнего индекса. Текст при этом располагается ниже базовой линии остальных символов строки и уменьшенного размера.

Пример тега sup:

X n + Y n = Z n

Пример тега sub:

СН3СНО + NH2OH = CH3C(=NOH)H + H2O

small, big

Описание:

Тег small уменьшает размер шрифта на единицу по сравнению с обычным текстом. В HTML размер шрифта измеряется в условных единицах от 1 до 7, средний размер текста, используемый по умолчанию, принят 3.Тег big увеличивает размер шрифта на единицу по сравнению с обычным текстом. В HTML размер шрифта измеряется в условных единицах от 1 до 7, средний размер текста, используемый по умолчанию, принят 3.

Пример тега small:

лев

Пример тега big:

BIG

blockquote, q

Описание:

Тег q используется для выделения в тексте цитат. Содержимое контейнера автоматически отображается в браузере в кавычках. Тег blockquote предназначен для выделения длинных цитат внутри документа. Текст, обозначенный этим тегом, традиционно отображается как выровненный блок с отступами слева и справа (примерно по 40 пикселов), а также с отбивкой сверху и снизу.

Пример тега q:

Чаще всего выход там, где был вход

Пример тега blockquote:

Lorem ipsum dolor sit amet, consectetuer adipiscing elit, sed diem nonummy nibh euismod tincidunt ut lacreet dolore magna aliguam erat volutpat. Ut wisis enim ad minim veniam, quis nostrud exerci tution ullamcorper suscipit lobortis nisl ut aliquip ex ea commodo consequat.

ins, del

Описание:

Тег ins предназначен для выделения текста, который был добавлен в новую версию документа. Подобное форматирование позволяет отследить, какие изменения в тексте документа были сделаны.Тег del используется для выделения текста, который был удален в новой версии документа. Подобное форматирование позволяет отследить, какие изменения в тексте документа были сделаны.

Пример тега ins:

подход разработчиков этого браузера был изменен

Пример тега del:

этот недостаток был устранен

abbr

Описание:

Тег abbr указывает, что последовательность символов является аббревиатурой.

Пример тега abbr:

CGI

address

Описание:

Тег address предназначен для хранения информации об авторе и может включать в себя любые элементы HTML вроде ссылок, текста, выделений и т.д.

Пример тега address:

Этот партал зделан сабствинаручно веб-дизайнером Ваней Шапочкиным в праграме Блакнот. Если вы хатите взять маи статьи к сибе на сайт то спрасите мое разришение.

nobr, wbr

Описание:

Тег nobr уведомляет браузер отображать текст без переносов.Тег wbr указывает браузеру место, где допускается делать перенос строки в тексте, если этого требует ширина родительского элемента.

Пример тега nobr:

Lorem ipsum dolor sit amet, consectetuer adipiscing elit

Пример тега wbr:

метоксихлордиэтиламинометилбутиламиноакридин
